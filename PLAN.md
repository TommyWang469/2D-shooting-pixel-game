# Pixel Dungeon Blaster — Plan & Devlog

A Soul Knight–style top-down twin-stick roguelite built in **Godot 4.7** (GDScript).
Move with WASD, aim with the mouse, shoot to kill enemies, collect coins & hearts,
upgrade your weapon, gain lives from kills, survive escalating waves.

- **Engine:** Godot 4.7 stable
- **Project folder:** `new-game-project/`
- **Language:** GDScript
- **Art:** real pixel-art PNG sprite sheets generated by `tools/gen_art.py` (PIL)

---

## Design Summary

**Core loop:** spawn in a room → move + aim + shoot → kill enemies → they drop coins &
occasional hearts → clear a wave → next wave is harder → spend coins at a weapon crate to
upgrade → survive. Die → game over → press R to restart.

**"More lives by killing enemies":** every 10 kills grants +1 max HP and a full heal;
hearts also drop occasionally from enemies.

**Weapon upgrades (3 tiers):** basic blaster → faster fire + more damage → triple-shot spread.
Bought by walking into a weapon crate when you have enough coins.

---

## Systems / Architecture

Each system is its own scene + script so it can be understood and tested in isolation.

| File | Responsibility |
|---|---|
| `autoload/game_manager.gd` | global state: coins, score, wave, kills; grants HP milestones; game-over/restart signals |
| `scenes/player/player.gd` | CharacterBody2D movement, mouse aim, HP, fires current weapon |
| `scenes/weapon/weapon.gd` (Resource) | fire rate, damage, bullet speed, spread, tier upgrades |
| `scenes/bullet/bullet.gd` | travels, collides, applies damage, frees self |
| `scenes/enemy/enemy.gd` | base: HP, hit flash, death puff, drops |
| `scenes/enemy/slime.gd` / `bat.gd` | AI variants (slow chaser / fast erratic) |
| `scenes/pickup/pickup.gd` | coin / heart / weapon-crate behaviors |
| `scenes/hud/hud.gd` | hearts, coin count, wave number, game-over panel |
| `scenes/spawner/spawner.gd` | wave timing, escalating enemy counts |
| `scenes/main/main.gd` | room + camera + wiring |

## Art Assets (generated by `tools/gen_art.py`)

- `player.png` — 16×16, 8 frames (4 idle + 4 walk)
- `slime.png` — 16×16, 4 frames (3 bounce + 1 death)
- `bat.png` — 16×16, 3 frames (2 flap + 1 death)
- `bullet.png` / `enemy_bullet.png` — 8×8
- `muzzle.png` — 8×8, 2 frames
- `coin.png` — 8×8, 4 spin frames
- `heart.png` — 8×8
- `crate.png` — 16×16
- `tiles.png` — floor + wall tiles (16×16 each)

All imported with **nearest** filtering (crisp pixels).

### 2026-07-17 — Polish pass (major)
Turned the vertical slice into a thoughtful, juicy roguelite.

**Game feel**
- `Juice` autoload: trauma-based **screen shake** (`camera.gd` on the player camera) + **hit-stop** (time-freeze on impactful hits).
- **Knockback** on enemies (bullets) and player (on hurt); i-frames + blink.
- **Muzzle flash**, **impact sparks** + **death particle bursts** (`fx/burst`), **floating combat text** (`fx/floating_text`) for damage / coins / weapon names / "LIFE UP!".
- **Drop shadows** under player and all enemies; **coin/heart magnet**.

**Audio** (procedural, `tools/gen_audio.py` — pure-Python WAV synth)
- SFX: shoot, hit, hurt, die, coin, heart, upgrade, wave, dash, gameover, click.
- Looping chiptune **music** track. `Audio` autoload with a pooled player + random pitch.

**Gameplay depth**
- **Dash** (Space / RMB): burst + i-frames + afterimage trail + cooldown meter.
- **Weapon system overhaul** — distinct weapons (Blaster, SMG, Triple, Shotgun, Piercer, Cannon) with pierce / knockback / colored bullets.
- **Treasure chests** swap you to a new, more powerful weapon.
- **Ranged Mage** enemy (keeps distance, casts `enemy_bullet`s).
- **Chapters → 3 combat rooms + a Boss room.** Boss has a health bar and cycles ring / summon / charge attacks; each chapter is harder and retinted. **Portals** advance rooms.

**Screens / UI**
- **Title screen** (new entry scene) with bobbing hero + music.
- **Pause** (Esc). Polished **HUD**: hearts, coin icon+count, weapon, dash meter, chapter/room, animated **room/boss banners**, boss bar, **low-HP red vignette**, game-over with run stats.
- **Atmosphere**: dim `CanvasModulate`, four wall **torches** with flame particles + additive glow.

New architecture nodes: `autoload/{audio,juice}.gd`, `scenes/dungeon/dungeon.gd`,
`scenes/enemy/{mage,boss}.*`, `scenes/pickup/{chest,portal}.*`, `scenes/fx/*`,
`scenes/title/*`, `scenes/bullet/enemy_bullet.*`, `scenes/player/camera.gd`.

**Verified:** clean headless import; 550-frame combat run + a dedicated smoke test
exercising Mage / Boss (damage→death→signals) / chest weapon-swap / portal — all 0 errors.
